# power_logger

A well log tool on flutter, support Dio and custom type data.

## Getting Started

### Usage

* init logger

```dart
PowerLogger.start(context);
```

* add data to logger

```dart
LoggerData.addData(data);
```

## WARNING

*only tested on flutter `1.22.5`,`1.22.6`
